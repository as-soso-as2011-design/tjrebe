# GitHub Pages جاهز

الملفات الأساسية:
- index.html
- teacher.html
- student.html
- teacher-report.html
- student-report.html

## الرفع إلى GitHub
1. افتحي المستودع في GitHub.
2. ارفعي كل الملفات الموجودة داخل هذا المجلد إلى جذر المستودع.
3. من Settings > Pages:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /(root)
4. احفظي الإعدادات.
5. الرابط سيفتح من index.html تلقائيًا.

## ملاحظات
