هذه النسخة فيها فصل كامل بين الجوال وبين الآيباد/الكمبيوتر:

- index.html / teacher.html / student.html / supervision.html = نسخة الآيباد والكمبيوتر
- mobile-index.html / mobile-teacher.html / mobile-student.html / mobile-supervision.html = نسخة الجوال البنكية
- التحويل يتم تلقائياً حسب عرض الجهاز
- إذا أردت فتح نسخة الكمبيوتر من الجوال مؤقتًا أضف ?desktop=1
- إذا أردت فتح نسخة الجوال من الكمبيوتر مؤقتًا أضف ?mobile=1
